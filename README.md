# Sistema de Gestión Integral "El Mirador" - Prototipo Funcional

## 🏢 Descripción del Proyecto
Este repositorio contiene los entregables correspondientes a la Evaluación 3 de Arquitectura de Software. El proyecto consiste en el despliegue y validación del prototipo funcional para el sistema de gestión del Condominio "El Mirador", abarcando los módulos operativos de Analítica, Administración, Conserjería y Residente.

## 🛠️ Herramientas Utilizadas
* **Prototipo Funcional:** Desarrollo e implementación interactiva del sistema.
* **Calidad y Pruebas:** Microsoft Excel (Plantillas de Casos de Prueba basadas en la norma ISO/IEC 25010 y Evaluación Heurística de Nielsen).
* **Control de Versiones:** Git y GitHub.

## 🔀 Práctica de Control de Versiones
Para la gestión de este proyecto se ha implementado la práctica de **Versionamiento Semántico (SemVer)** bajo la nomenclatura `Mayor.Menor.Parche` (ej. `v1.0.0`). 

**Justificación técnica:**
Se eligió el Versionamiento Semántico integrado con GitHub porque permite al equipo mantener un historial claro, cronológico e inalterable del desarrollo. 
* Un cambio **Mayor** (`v1.0.0` a `v2.0.0`) justifica refactorizaciones completas de la arquitectura.
* Un cambio **Menor** (`v1.0.0` a `v1.1.0`) refleja la adición de nuevos módulos funcionales o entregables técnicos.
* Un cambio de **Parche** (`v1.1.0` a `v1.1.1`) se utiliza para correcciones de bugs, optimización de código o ajustes menores.
Esta práctica facilita la iteración continua y previene la pérdida de información en el entorno de trabajo colaborativo.
