##  Descripción del Problema
El Condominio "El Mirador" enfrenta graves problemas de administración manual, lo que ha generado una alta morosidad, ineficiencia en el control de accesos e inseguridad para los copropietarios.

## Solución (Prototipo)
Se diseñó un sistema web integral que automatiza la gestión financiera, el control de bitácoras y la comunicación comunitaria.
